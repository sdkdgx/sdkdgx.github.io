节假日青岛游-小tips

###关于节日两三天适合来青岛玩的地方~
话说今年青岛市里有一条地铁线出行也方便了不少hhh～
我推荐这几个地方～

###1、八大关与海
关键字：樱花、海景、欧式建筑
推荐路线：八大关-花石楼建筑-第二海水浴场

红墙绿树，碧海蓝天，八大关景区可以说是欧式特色建筑与花鸟树木的融合，与之临近的就是沙滩
海边，花开一路，可以说是青岛数一数二的游玩景点啦~~

###2、青岛印象
关键字：中西结合，青岛特色
推荐路线：青岛火车站-栈桥-中山路-天主教堂-劈柴院

青岛的火车站离海边只有几分钟的路程，从海边栈桥沿着青岛特色街之一的中山路一路向北，从欧
式建筑转到中式建筑，沿途有西式教堂，也有中式特色街，沿街亦有几家颇具特色风格的小店～

###3、五四青岛
关键字：五月的风、火炬、帆船、情人坝、灯塔
推荐路线：五四广场-奥帆中心

五四运动高中历史课本都讲过吧，这有个地标建筑就叫五月的风，沿着海岸就能看到不远处奥帆基
地的奥运火炬，那边有帆船、海鸥、游艇，笔直的情人坝尽头是一座白色的灯塔～

户外景点我就推荐这三个啦，接下来，还要讲一些坑：
敲黑板:假日慎重选择人山人海的室内展馆和旅途劳顿的偏远地区
青岛的还两个海洋馆–海底世界和极地海洋世界，上次国庆我和小伙伴去极地海洋世界，真的是哇，海洋动物在哪里，海豹、海豚都淹没在了人海中，慎选慎选。。。
另外，崂山虽是5A级景区，但也是比较远来回就四五个小时，要爬的话就要单独拿一天时间，加上爬山累是免不了啦，感觉不适合两三天的情况~~

关于天气环境，青岛海边风大，昼夜温差会大一点，注意带外套，去海边的话也可以带便携拖鞋.
关于当地小吃，海鲜和啤酒是亮点，特色食物有辣炒蛤蜊、青岛大虾、鱿鱼、蒸饺、锅贴.
关于住宿，这边青年旅社挺多哒，几个小伙伴可以考虑住青旅，全国连锁的酒店旅馆也都有.
以上是文字版的简单介绍，可以稍作参考～
